# Port-Scanner
A port scanner created with Python 3.10.2.

Third party modules:

  • IPy
  • Colorama

Installation:

1. Just clone this repository or download this as a zip
  
  git clone https://github.com/Anz1x/Port-Scanner

2. Install the modules IPy and Coloroma if not installed

  pip3 install IPy
  pip3 install IPy
